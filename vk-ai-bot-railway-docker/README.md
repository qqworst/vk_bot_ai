# VK AI Bot (Gemini) — Railway (Docker) Ready

## Deploy steps

1. Upload this project to GitHub.
2. Go to Railway → New project → Deploy from GitHub.
3. Railway will detect Dockerfile automatically.
4. Add variables in Railway → Variables:
   - VK_API_TOKEN
   - GEMINI_API_KEY
   - GROUP_ID = 233981911
   - CONFIRMATION_STRING = a6517b79
5. Copy service URL → VK → Callback API → Add server → Confirm.

Bot is ready!
